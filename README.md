Jangan lupa untuk melakukan push ke repository ini.
  

*This is a Next.js project bootstrapped with create-next-app.*

## Prasyarat

Sebelum memulai, pastikan komputer Anda telah terinstall:

-   [Node.js](https://nodejs.org/) (Versi 18 atau lebih baru direkomendasikan)
-   [Git](https://git-scm.com/)

## Cara Menjalankan Project (Getting Started)

Ikuti langkah-langkah di bawah ini untuk menjalankan project di komputer lokal Anda (Localhost).

### 1. Clone Repository

Salin project ini ke komputer lokal Anda dengan perintah:

```bash
git clone https://github.com/regenerasi8eh2025/trial-of-true-colors.git
```

### 2. Masuk ke Direktori

Masuk ke folder project yang baru saja di-clone:

```bash
cd trial-of-true-colors
```

### 3. Install Dependencies
Install semua library yang dibutuhkan. Anda bisa menggunakan npm, yarn, atau pnpm:

```bash
npm install
# atau
yarn install
# atau
pnpm install
```

### 4. Konfigurasi Environment Variables
Konfigurasi `.env`

### 5. Jalankan Development Server
Jalankan server lokal dengan perintah:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
```

Buka http://localhost:3000 pada browser Anda untuk melihat hasilnya.
